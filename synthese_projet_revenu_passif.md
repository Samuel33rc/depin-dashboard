# Synthèse - Projet Revenu Passif

## Les 3 Angles Creusés

### 1. DePIN (Réseaux Décentralisés Physiques)

| Projet | Ce que tu fournis | Revenu potentiel |
|--------|------------------|------------------|
| Helium (HNT) | Hotspot WiFi/5G | ~$10-50/mois |
| Hivemapper | Dashcam + carto | ~$100-400/mois |
| DIMO | Data voiture (OBD) | ~$20-100/mois |
| Render (RNDR) | GPU pour IA/rendu | Variable |
| Filecoin | Stockage | Variable |
| Grass (AI) | Bandwidth internet | ~$5-50/mois |

Points clés :
- Marché en pleine explosion ($150M revenus/mois en Jan 2026)
- Certains projets commencent à payer en réel (pas que des tokens)
- Certains nécessitent hardware (pas 0€), d'autres juste un PC/téléphone

### 2. API Gluing (Connecter des services)

L'idée : Créer des automations qui connectent 2 outils qui ne communiquent pas, pour résoudre un problème spécifique.

Exemples de "trous" :
- Connecter un outil niche à Shopify/WordPress
- Automatiser un flux B2B entre 2 logiciels peu connus
- Synchroniser des données entre une app mobile et un CRM obscure

Outils gratuits pour construire :
- n8n (open source, très puissant)
- Activepieces (alternatif Zapier)
- Pipedream (intégrations API)
- Make (ex-Integromat)

Comment monetiser :
- Abonnement mensuel pour l'automation
- Service de configuration one-time
- Template/revente d'automations

### 3. Micro-Niche B2B (Logiciel très ciblé)

Le principe : Résoudre 1 problème pour 1 métier spécifique — pas "AI pour tous".

Exemples de niches silencieuses :
- Outil pour notaires
- Outil pour réparateurs de vélos
- Outil pour courtiers en assurance artisanale
- Plugin pour un logiciel niche (ex: plugins pour Airtable, Notion, monday.com)

Pourquoi ça marche :
- Concurrence quasi nulle
- Clients fidèles (switching cost élevé)
- Prix élevés possibles car ROI clair pour le client
- Boring = stable

Pour construire (sans coder) :
- Bubble (web apps)
- Glide (apps mobile)
- Softr (portails clients)
